# вЪЩпЄП Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/AI_JOB_TRACKER.git
```

Go to the project folder:

```bash
cd AI_JOB_TRACKER
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment:

### Windows

```bash
venv\Scripts\activate
```

### macOS/Linux

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Download the SpaCy model:

```bash
python -m spacy download en_core_web_sm
```

Run the application:

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
